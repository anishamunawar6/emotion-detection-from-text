# AI-Based Emotion Detection from Text

## Overview
This project is an AI-based text analysis system that detects emotions from user input text. It classifies emotions such as happy, sad, angry, and more using Python.

---

## Features
- Detects emotions from text messages  
- Simple and user-friendly implementation  
- Uses dataset (CSV file) for processing  
- Built using Python  

---

## Project Files
- emotion_detection_text.py → Main Python code  
- processed_emotions.csv → Dataset file  
- presentation/ → Project presentation files  
- users/ → User-related data folder  

---

## Requirements
- Python 3.x  
- pandas library  

Install dependencies:
```bash
pip install pandas
```

---

## How to Run
1. Clone or download the repository  
2. Open terminal in project folder  
3. Run the script:

```bash
python emotion_detection_text.py
```
